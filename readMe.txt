README File

Fonctionnement du projet en lançant le code grâce à Qt :

Télécharger le dossier "Mini-games" et "Images"

Les mettre dans le même dossier.

Dans QT maintenant : 

- Ouvrir le fichier .pro
- exécuter Qmake
- cliquer sur la flèche verte en bas à gauche "exécuter"

Concernant le makefile, il y en a un automatiquement crée par Qt, qui se trouve dans le dossier "Images" : c'est le fichier "makefile release".

Enjoy :)
